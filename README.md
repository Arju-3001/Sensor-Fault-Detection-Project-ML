# Sensor Fault Detection using Machine Learning

## Project Overview
This project detects sensor faults using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask

## ML Algorithm
- Random Forest Classifier

## Steps
1. Data preprocessing
2. Train model
3. Save model
4. Predict sensor fault

## Run
pip install -r requirements.txt
python train.py
python app.py
